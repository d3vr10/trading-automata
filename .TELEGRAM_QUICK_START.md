# Telegram Bot - Quick Start (5 Minutes)

## Step 1: Create Bot Token (2 minutes)

```bash
1. Open Telegram
2. Search: @BotFather
3. Type: /newbot
4. Name it: Trading Bot
5. Username: trading_bot_abc
6. Copy the token you receive
```

**You'll get something like:**
```
1234567890:ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghi
```

## Step 2: Get Chat ID (2 minutes)

```bash
1. Search for your bot: @trading_bot_abc
2. Click START
3. Visit this in browser (replace TOKEN):
   https://api.telegram.org/bot<TOKEN>/getUpdates
4. Find: "chat": {"id": YOUR_ID_HERE}
```

**You'll get something like:**
```
987654321
```

## Step 3: Configure Bot (1 minute)

**Option A: Update .env**
```bash
cat >> .env << 'EOF'
TELEGRAM_TOKEN=1234567890:ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghi
TELEGRAM_CHAT_ID=987654321
EOF
```

**Option B: Update config/config.yml**
```yaml
app:
  telegram_token: "1234567890:ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghi"
  telegram_chat_id: "987654321"
```

## Step 4: Start Bot

```bash
pip install -r requirements.txt
python -m src.main
```

You should see:
```
INFO | ✅ Telegram bot initialized
```

And on Telegram:
```
🤖 Trading Bot Started
Environment: PAPER
Broker: ALPACA
```

## Step 5: Try Commands

```
/status  → Show bot status
/trades  → Last 10 trades
/metrics → Performance stats
/pause   → Stop trading
/resume  → Resume trading
/help    → Show commands
```

## Troubleshooting

| Problem | Solution |
|---------|----------|
| "Invalid token" | Check token format from BotFather |
| "Chat ID not found" | Send message to bot, then check getUpdates URL |
| "Bot not initialized" | Check TELEGRAM_TOKEN and TELEGRAM_CHAT_ID are set |
| "No alerts received" | Check if trades are actually executing |
