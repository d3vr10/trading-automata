# Quick Start

## Prerequisites

- Docker & Docker Compose
- Node.js 22+ (for local web-ui development)
- Python 3.13+ (for local engine/API development)

## 1. Environment Variables

```bash
cp .env.example docker/.env
```

Edit `docker/.env` and set the **required** values:

```env
# Required — generate with: python -c "from cryptography.fernet import Fernet; print(Fernet.generate_key().decode())"
FERNET_KEY=your-fernet-key

# Required — any random string, e.g.: openssl rand -hex 32
JWT_SECRET=your-jwt-secret

# Optional — leave empty to auto-generate root password on first boot
ROOT_USERNAME=root
ROOT_PASSWORD=

# Broker credentials (at least one)
ALPACA_API_KEY=your-key
ALPACA_SECRET_KEY=your-secret
```

## 2. Start the Stack

```bash
cd docker
docker compose up -d
```

Services:
- **PostgreSQL**: `localhost:5432`
- **Redis**: `localhost:6379`
- **API**: `localhost:8000` (docs at `/docs`)
- **Web UI**: `localhost:3000`

## 3. First Login

Check logs: `docker compose logs api | head -20`

Open `http://localhost:3000` and sign in.

## 4. Bot Configuration

Bots are defined in `config/bots.yaml`:

```yaml
bots:
  - name: "my-bot"
    enabled: true
    broker:
      type: alpaca
      environment: paper
    strategy:
      name: sigma_alpha_bull
    portfolio:
      allocated_capital: 10000
    fence:
      type: crypto
```

Restart the engine after changes: `docker compose restart trading-engine`

## 5. Production

```bash
cd docker
docker compose -f docker-compose.prod.yml up -d
```

Uses Traefik for TLS with Let's Encrypt. Set `DOMAIN` and `ACME_EMAIL` in `.env`.

## 6. Observability

Prometheus scrapes metrics from:
- API service: `http://api:8000/metrics`
- Trading engine: `http://trading-engine:8081/metrics`

Grafana is available at `http://localhost:3001` (admin/admin).

See [docs/OBSERVABILITY.md](docs/OBSERVABILITY.md) for metric definitions.

## API Documentation

With the API running, open `http://localhost:8000/docs` for the interactive Swagger UI.
