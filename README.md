## Architecture

```
trading-bot/
├── services/
│   ├── trading-engine/    # Python — bot orchestration, brokers, strategies
│   ├── api/               # FastAPI — auth, CRUD, WebSocket, metrics
│   └── web-ui/            # NextJS 16 — dashboard (API-only communication)
├── shared/alembic/        # Centralized database migrations
├── docker/                # Docker Compose (dev + prod)
└── config/                # Runtime YAML configs
```

## Stack

- **Trading Engine**: Python 3.13, asyncio, Alpaca/Coinbase SDKs
- **API**: FastAPI, SQLAlchemy (async), JWT + bcrypt, Fernet encryption
- **Web UI**: NextJS 16, React 19, shadcn/ui, Tailwind v4
- **Data**: PostgreSQL 17, Redis 7 (pub/sub + status cache)
- **Observability**: Prometheus, Grafana
- **Deployment**: Docker Compose, Traefik (production TLS)
